- js

## 基础框架（vue3与react17）

- 代码实现风格
  - react  class =>  函数式组件  最终返回jsx描述页面
  - vue 简单易用 sfc单文件模式定义组件  一系列辅助模板指令
- 数据更新机制
  - react immutable 调用特定函数
  - 响应式
- 代码重用性
  - react hooks重用性极高
  - composition API  类似react hooks系统
